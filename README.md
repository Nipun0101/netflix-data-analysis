# Netflix Data Analysis

This project analyzes the Netflix Movies and TV Shows dataset using Python.
The goal is to explore trends in genres, release years, and content distribution using data analysis and visualization.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset
The dataset contains information about Netflix movies and TV shows including:
- Title
- Director
- Cast
- Country
- Release Year
- Rating
- Duration
- Genre

## Analysis Performed
- Data Cleaning
- Handling Missing Values
- Genre Analysis
- Release Year Trends
- Data Visualization

## Key Insights
- Drama and International Movies are the most common genres on Netflix
- Netflix content increased rapidly after 2015
- Movies are more common than TV Shows on the platform
